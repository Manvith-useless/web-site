# Characters, Abilities, and Progression

## Cameleon (Redemption Campaign Protagonist)
- Identity: Illegal Meridan awakener; shape-shifts into any human form with proximity scan.
- Core Loop: Infiltrate using disguises, isolate targets, Echo Blade precision, moral choice cleanup.
- Ability Tree Branches:
  - Impostor: faster scan, extended disguise duration, suspicion dampening, social mimics.
  - Echo Duelist: parry windows, gap-closers, Echo Link single-target amplification.
  - Ghost: silent movement, non-lethal chain takedowns, light control gadgets.

## Devil (Antagonist/Playable in Corruption)
- Identity: Pact with Greed; consumes matter/energy to convert into raw power.
- Core Loop: Aggressive area denial, resource hoarding, corruption mechanics.
- Ability Tree Branches:
  - Devour: consume environment and enemies to stack power, volatile overflow blasts.
  - Greed’s Bargain: trade health/mercy for massive burst windows, collateral risk.
  - Tyrant: fear auras, crowd control, civilian manipulation.

## Zero (Weapons Savant, 554567)
- Identity: Government experiment; instantly masters any weapon.
- Core Loop: Adapt loadouts on the fly, counter boss patterns, weapon synergy.
- Ability Tree Branches:
  - Arsenal: swap weapon classes mid-combo, ammo conjure, stance memory.
  - Countermind: pattern recognition, perfect guard, parry projectiles.
  - Vanguard: gap-closers, team buffs, revive mastery.

## Eclipse (Shadow Manipulator, 554567)
- Identity: Master of Zero; manipulates shadows, illusions, and light occlusion.
- Core Loop: Zone control, clone misdirection, line-of-sight denial.
- Ability Tree Branches:
  - Umbral Arts: shadow clones, delayed strikes, darkness fields.
  - Eclipse Tide: pull/push via shadow currents, trap setups.
  - Night Sovereign: boss-tier control, light siphon, stealth countermeasures.

## Progression Systems
- XP from missions, stealth grades, and co-op synergy actions.
- Unlocks feed ability trees and gear crafting. Morality gates unique skills in both campaigns.
- Reputation influences intel access, merchant inventory, and safehouse upgrades.
